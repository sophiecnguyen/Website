DNA Project

C++ This program is a DNA analysis tool that validates DNA strands and lets users perform operations like calculating similarity, finding the best strand alignment, detecting mutations, transcribing DNA to RNA, generating reverse complements, and identifying coding frames. It’s all menu-driven, so the user just selects an option, inputs sequences, and the program outputs the corresponding analysis.

Lion King Game

C++ This program is a two-player text-based board game where players choose lion characters, follow different life paths, and take turns rolling dice to advance toward Pride Rock. Along the way they gain or lose pride points and stats, and once both reach Pride Rock their final points are tallied (with extra stats converted into points) to determine who becomes the Pride Leader, with the winner’s info saved to a file.

Dungeon Simulation

C++ This program implements a hash table to store character items, where each entry contains a character name, an item, its power, and a description. Users can insert items, search for characters, delete specific items, and print the table to see all entries. The program handles collisions internally and can report the total number of collisions, effectively combining hashing with item management for quick lookups and updates.
